Onjob API Backend

This is the backend API server for the Onjob platform. Please use the appropriate API endpoints to interact with the service.

Available Endpoints:

GET /api/jobs - Get all jobs
GET /api/jobs/:id - Get job by ID
GET /api/jobs/categories - Get job categories
GET /api/jobs/suggestions - Get job suggestions
POST /api/users/login - User login
POST /api/users/register - User registration

This API server is designed to work with the Onjob frontend application.
